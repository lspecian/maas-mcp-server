# MCP Inspector Testing Support

This directory contains documentation and resources for manual testing of the MAAS MCP Server using the MCP Inspector tool. The MCP Inspector allows interactive testing of the server against the Model Context Protocol (MCP) specification.

## Contents

- [Setup Guide](setup_guide.md): Instructions for setting up the MCP Inspector testing environment
- [Testing Procedures](testing_procedures.md): Step-by-step procedures for testing the MAAS MCP Server
- [Test Scenarios](test_scenarios.md): Specific test scenarios for different functionality areas
- [Troubleshooting](troubleshooting.md): Common issues and their solutions
